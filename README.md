# دوره هوش مصنوعی مولد

این مخزن به‌عنوان مرجع رسمی دوره «هوش مصنوعی مولد» در مرکز فناوری اطلاعات شهرستان گرگان ایجاد شده است. تمامی منابع آموزشی، مستندات، سرفصل‌ها و فایل‌های دوره از طریق این مخزن در دسترس شرکت‌کنندگان قرار می‌گیرد.

## سرفصل‌های آموزشی دوره

- **مهندسی پرامپت و آشنایی با مدل‌های زبانی بزرگ (LLMها)**  
  اصول، الگوها و تکنیک‌های تعامل مؤثر با مدل‌های مولد.

- **مباحث پیشرفته RAG (Retrieval-Augmented Generation)**  
  طراحی، پیاده‌سازی و بهینه‌سازی سیستم‌های ترکیبی بازیابی–تولید برای ارائه پاسخ‌های دقیق و مبتنی بر داده.

- **طراحی و توسعه دستیارهای سفارشی GPT**  
  ساخت GPTهای اختصاصی برای اتوماسیون وظایف تخصصی و کاربردهای سازمانی.

- **ساخت AI Agents با ChatGPT و Zapier**  
  اتصال مدل‌های زبانی به ابزارهای خارجی، طراحی عامل‌ها و ایجاد جریان‌های کاری هوشمند.

- **تحلیل داده با ChatGPT و Excel**  
  استفاده از مدل‌های زبانی در کنار ابزارهای تحلیلی برای تولید گزارش‌های داده‌محور و ساده‌سازی فرآیندهای تحلیل.

- **Data Storytelling و تصمیم‌گیری مبتنی بر داده**  
  تبدیل داده‌های خام به روایت‌های قابل فهم برای مدیران، استخراج بینش کاربردی و پشتیبانی از تصمیم‌سازی هوشمند.
