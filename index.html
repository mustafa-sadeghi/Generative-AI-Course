دوره تخصصی هوش مصنوعی مولد (Generative AI)

مرکز فناوری اطلاعات شهرستان گرگان – تمرکز ویژه بر مهندسی پرامپت و به‌کارگیری عملی LLMها

این دوره برای افرادی طراحی شده است که می‌خواهند فراتر از استفاده سطحی از ChatGPT و ابزارهای مشابه بروند و مهارت مهندسی پرامپت (Prompt Engineering) را به‌صورت عمیق، ساختاریافته و کاربردی یاد بگیرند. علاوه بر مباحث فنی GenAI، تمرکز اصلی دوره روی این است که چگونه با طراحی پرامپت‌های حرفه‌ای، مدل‌های زبانی را به سمت خروجی‌های دقیق، قابل‌کنترل و متناسب با نیازهای کسب‌وکار هدایت کنیم.

در این دوره، مهندسی پرامپت فقط یک فصل تئوری نیست؛ در تمام طول دوره، روی طراحی پرامپت برای سناریوهای واقعی کار می‌کنیم: از نوشتن پرامپت برای تحلیل داده و تولید گزارش، تا طراحی Prompt Pipeline برای RAG و ساخت Agentهای هوشمند. هدف این است که شرکت‌کننده بتواند بعد از دوره، خودش سیستم‌های پرامپت‌محور برای سازمان یا پروژه‌های شخصی‌اش طراحی کند.

سرفصل‌های آموزشی دوره

  • مبانی هوش مصنوعی مولد و مدل‌های زبانی بزرگ (LLMها)
    آشنایی با مفاهیم پایه، معماری مدل‌های زبانی و نقش پرامپت در هدایت رفتار مدل.
  • مهندسی پرامپت (Prompt Engineering) در عمل
    الگوهای استاندارد پرامپت‌نویسی، طراحی پرامپت‌های چندمرحله‌ای، زنجیره‌سازی پرامپت‌ها (Prompt Chaining)، تعریف نقش (Role Prompting)، مدیریت Context و کاهش خطاهای مدل.
  • مباحث پیشرفته RAG (Retrieval-Augmented Generation)
    ترکیب تکنیک‌های جست‌وجو و پرامپت‌نویسی برای ساخت سیستم‌هایی که بر اساس اسناد و داده‌های واقعی پاسخ می‌دهند، طراحی پرامپت‌های مخصوص RAG و کنترل منبع پاسخ‌ها.
  • طراحی و توسعه دستیارهای سفارشی GPT
    تبدیل مهارت پرامپت‌نویسی به دستیارهای پایدار و قابل‌استفاده مجدد، تعریف دستورالعمل‌ها، محدودیت‌ها و طراحی Prompt Template برای سناریوهای سازمانی.
  • ساخت AI Agents با ChatGPT و Zapier
    استفاده از پرامپت برای تعریف رفتار Agent، اتصال به ابزارها و سرویس‌های بیرونی، و طراحی گردش‌کارهایی که با ترکیب منطق، داده و اتوماسیون، کارهای واقعی را انجام می‌دهند.
  • تحلیل داده، گزارش‌نویسی و Data Storytelling با پرامپت‌نویسی هدفمند
    طراحی پرامپت‌هایی برای تحلیل داده در کنار Excel، ساخت گزارش‌های مدیریتی، خلاصه‌سازی هوشمند و تبدیل خروجی خام مدل به روایت قابل‌فهم برای تصمیم‌گیران.

فصل‌های منتشرشده دوره

فصل‌های آموزشی به‌صورت تدریجی منتشر می‌شوند. هر فصل شامل توضیحات مفهومی، مثال‌های عملی و نمونه‌پرومپت‌هایی است که می‌توانید مستقیماً در پروژه‌ها و محیط کاری خود استفاده کنید.
